# Warehouse_mobile

Mobilní multiplatformní aplikace (Android OS, iOS) pro práci se sklady materiálu ve skautISu vznikla jako bakalářská práce na FI MU v Brně, jejímž autorem je [Honza Crha](https://is.muni.cz/th/ibkmp/).

Pro vývoj aplikace byl zvolen framework [NativeScript](https://www.nativescript.org), který umožňuje build nativních aplikací pro obě majoritní platformy současnosti.

# Přehled dostupných funkcí aplikace

* přihlášení přes skautIS
* načtení dostupných skladů
* vyhledání položek skladu dostupných ve vlastním skladu
* vyhledání položek, které jsou dostupné k vypůjčení ze skladů dalších jednotek
* rezervace položek (1. krok - žádost o rezervaci)
* inventarizace majetku (vyžaduje již založenou inventarizaci ve skautISu)
